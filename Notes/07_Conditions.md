# JavaScript Conditional Statements

## Introduction

Conditional statements allow a program to make decisions based on different conditions.

Instead of executing every line of code, JavaScript can decide which block of code should run depending on whether a condition is **true** or **false**.

Conditional statements are one of the most important concepts in programming because they make applications interactive and dynamic.

For example:

- Login only if the password is correct.
- Show "Eligible" if a student's mark is above 50.
- Display "Good Morning" before 12 PM and "Good Evening" after 12 PM.
- Show different pages based on the user's role.

JavaScript provides several ways to perform decision-making:

1. **if Statement**
2. **if...else Statement**
3. *switch Statement**
